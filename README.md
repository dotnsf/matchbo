# Match-Bo


## Overview

Auto solver for **「マッチ一本だけ動かして、正しい式にしてください」**


## How to play

- Input incorrect formula in robot's mouth, and submit it.

- You will find corrected formula in robot's body.


## Specification

- [x] More than 2 '='s.

- [ ] Fraction

- [ ] Brackets


## Rule

Based on [BS8 **クイズ!脳ベル SHOW**](https://www.bsfuji.tv/noubellshow/)
